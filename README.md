# 螢幕劃分分析

- 螢幕縮放發現 中螢幕和大螢幕布局是一致的，因為列定義為col-md- 就可以了，md是>=970以上的
- 螢幕縮放發現 小螢幕布局發生變化，因此需要為小螢幕根據需求改變布局
- 螢幕縮放發現 超小螢幕布局又發生變化，因此需要為超小螢幕根據需求改變布局
- 策略： 先布局md以上的pc端布局，最後根據實際需求再修改小螢幕和超小螢幕的特殊布局樣式
